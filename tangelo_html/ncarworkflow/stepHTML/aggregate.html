Analyze Data

Node

The Node this analysis pulls data from.

Aggregate Type

The type of aggregate you want to use.

Time Interval

The time interval over which to aggregate.

Output Date

The date of which to set aggregate values. Ex: If the output date is set to "start" for a monthly aggregate, the aggregate values will be set at the first over every month in the output data.

Cyclic Aggregation

Aggregate over years if there is more than one year in input data.
