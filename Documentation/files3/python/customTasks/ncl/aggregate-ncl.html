Aggregate

A NCL script that aggregates NetCDF data by different time intervals and finding the min, max, or mean of the data.

Attributes

filenameThe name of the NetCDF file of which to aggregate.
intervalThe time interval to aggregate over.
methodThe method of aggregation (Min, Max, Mean).
outtimeThe outtime of the aggregation values.
cyclicWhether or not to aggregate over years.

Output Aggregation

Creates the output NetCDF file in the corresponding workflow and adds the appropriate metadata.

Parameters

outtimeThe outtime within the interval to set the aggregate values in the output file.
cyclicWhether or not to aggregate over years.
variableThe variable over which to aggregate, obtained from the NetCDF file’s MainVariable attribute.
widThe workflow id, passed from the python task.
tidThe task id, passed from the python task.

Returns

An aggregate NetCDF file in the workflow folder with the name tid_aggregate.nc.  The metadata of the new file contains the updated time, time_bnds, date, and date_bnds.
